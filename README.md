Este repositorio contiene mis apuntes, retos de lógica y proyectos prácticos desarrollados a lo largo del curso de JavaScript (ES6+) de JonMircha.

## 🛠️ Tecnologías y Conceptos Aplicados
* JavaScript Moderno (ES6+)
* Manipulación del DOM y Eventos
* Programación Asíncrona (Promesas, Async/Await)
* AJAX y Consumo de APIs (Fetch, Axios)

## 📁 Estructura del Proyecto
* `/ejercicios`: Soluciones a los retos de lógica de programación.
* `/dom`: Prácticas de interactividad y maquetación web dinámica.
